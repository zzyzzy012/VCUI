# VueEle

## 项目简介

这是一个仿照ElementPlus的组件库，基于Vue开发，轻量、高度自定义，服务于企业级的后台管理系统项目。
在线项目地址：https://vcui-zzys-projects-520709aa.vercel.app/

## 项目运行

```
npm install
npm run dev
npm run build
npm run lint
npm run docs:dev  #文档库运行
```
